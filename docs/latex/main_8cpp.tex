\doxysubsection{Detailed Description}
Основной файл программы для сравнения алгоритмов сортировки 

\begin{DoxyAuthor}{Author}
Кирилл Моисеенко 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2023
\end{DoxyDate}
Программа считывает данные о футболистах из CSV-\/файлов, выполняет сортировку четырьмя разными алгоритмами, измеряет время выполнения и сохраняет результаты.
